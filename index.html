Pokédex iOS

Lightweight, Virtual iPhone Pokédex

This project is ongoing, and far from complete. If you would like to contribute, see the Contribution section.

Pokédex–iOS is a versatile, lightweight virtual Pokédex for your iPhone or iPod Touch.

Contribution

If you would like to contribute a feature to this project, email me first. Once I approve of your idea, you can create a pull request with your changes. Changes made in the Xcode project should be made to the pokedex-xcode repository, because that contains the source code for this project. If you want to contribute to artwork / sprites, use the pokedex-artwork repository.
